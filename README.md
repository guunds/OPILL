# OPiLL Protocol

DeFi web app on OP_NET (Bitcoin L2).

## Deploy
1. Upload to Vercel via GitHub or drag-drop
2. All images are embedded as base64 inside index.html (no CDN needed)
3. The `images/` folder is a backup of the NFT artwork
