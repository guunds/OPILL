
OPILL NFT MINT TEMPLATE

Simple Web3 NFT mint page for testnet.

FILES
index.html -> main mint page
style.css -> UI style
app.js -> wallet connect + mint logic

SETUP

1. Deploy your NFT contract
2. Open app.js
3. Replace:

const contractAddress = "YOUR_NFT_CONTRACT_ADDRESS"

4. Upload files to:
- Vercel
- Netlify
- GitHub Pages

FEATURES
- Connect MetaMask
- Mint NFT
- Clean UI
- Testnet ready
